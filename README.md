# ScreenPilot (Desktop Screen Time Control Prototype)

ScreenPilot is a frontend prototype for a desktop screen-time control app with:

- Permission request modal for activity-data access.
- Analytics with day/week/month/custom range filters.
- Usage chart + flowchart visualization.
- App blocker/time limit manager with exceeded-limit alerts.
- Email signup/login mock and multi-device control panel.
- Theme integration (system default, light, dark) with red accent UI.

## Run locally

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.
